HERITAGE

Biological Statistics

Animal traits, hybrid heritage, Quimera percentage

🐾 Animal Trait Score (ATS)

Acquired through: magic, bloodline, or transformation. Roll d10 if applicable.

Score Description
0 No animal traits
1-3 Minor (ears, eyes, teeth)
4-6 Moderate (partial limbs, senses)
7-9 Major (full hybrid appearance)
10 Complete transformation
Animal Buffs: +2 to one physical stat (Might/Finesse/Endurance) per trait tier
Animal Debuffs: -1 to one mental stat (Intellect/Insight/Presence) per trait tier

Example: Wolf traits (ATS 6): +2 Finesse, +2 Might, -1 Presence.

🧬 Hybrid Trait Score (HTS)

Born with mixed heritage. Roll d10.

Score Description
0 Pureblood
1-3 Distant hybrid (¼ heritage)
4-6 Half-blood
7-9 Dominant hybrid (¾ heritage)
10 Perfect hybrid blend
Hybrid Buffs: +1 to two different stats per tier
Hybrid Debuffs: -1 to one stat per tier (opposite of buffs)

Example: Elf-human hybrid (HTS 5): +1 Finesse, +1 Insight, -1 Endurance.

📊 Quimera Percentage (Q%)

Q% = [ATS × 5] + [HTS × 3] + [Animent/10]

Chance that character's offspring will be quimera (mixed-species).

  • 0-20%: Normal offspring
  • 21-50%: Possible minor quirks
  • 51-80%: Likely quimera traits
  • 81-100%: Guaranteed quimera offspring

⚖️ Social Implications

High Q% causes social stigma: -2 to Presence in conservative societies. Magical birthmarks glow when using affinity element (can't be hidden).
